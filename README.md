A collection of Appmixer actions for seven.

## Installation

```shell
git clone https://github.com/seven-io/appmixer
```

```shell
cd appmixer/appmixer.seven/appmixer/seven
```

```shell
appmixer publish
```

### Set API Key

Add an environment variable called `SEVEN_API_KEY`.
